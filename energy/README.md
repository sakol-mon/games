# เกมจับคู่พลังงานทดแทน 🌿

เกมเปิดไพ่จับคู่ระหว่างชื่อพลังงานทดแทนภาษาไทยกับรูปภาพ

## 📁 โครงสร้างไฟล์

```
project/
├── index.html          # ไฟล์ HTML หลัก
├── script.js           # ไฟล์ JavaScript (แยกออกจาก HTML)
├── cards.json          # ไฟล์ JSON เก็บข้อมูลการ์ด
├── images/             # โฟลเดอร์สำหรับเก็บรูปภาพ
│   ├── cover.png      # รูปด้านหลังการ์ด
│   ├── 1.png          # รูปพลังงานแสงอาทิตย์
│   ├── 2.png          # รูปพลังงานลม
│   ├── 3.png          # รูปพลังงานน้ำ
│   ├── 4.png          # รูปพลังงานชีวมวล
│   ├── 5.png          # รูปพลังงานความร้อนใต้พิภพ
│   └── 6.png          # รูปพลังงานไฮโดรเจน
└── README.md          # ไฟล์นี้
```

## 🎮 คู่ที่ต้องจับ

1. **พลังงานแสงอาทิตย์** (Solar Energy) ↔ 1.png
2. **พลังงานลม** (Wind Energy) ↔ 2.png
3. **พลังงานน้ำ** (Hydropower) ↔ 3.png
4. **พลังงานชีวมวล** (Biomass) ↔ 4.png
5. **พลังงานความร้อนใต้พิภพ** (Geothermal Energy) ↔ 5.png
6. **พลังงานไฮโดรเจน** (Hydrogen Energy) ↔ 6.png

## 🚀 วิธีการติดตั้ง

1. วางรูปภาพในโฟลเดอร์ `images/`:
   - **cover.png** - รูปด้านหลังการ์ด (ปกเกม)
   - **1.png ถึง 6.png** - รูปพลังงานแต่ละประเภท
2. เปิดไฟล์ `index.html` ในเว็บเบราว์เซอร์

## 📝 วิธีการเล่น

1. คลิกการ์ดเพื่อเปิด
2. เปิดการ์ด 2 ใบต่อครั้ง
3. ถ้าการ์ดจับคู่กัน (ชื่อตรงกับรูป) การ์ดจะค้างเปิดอยู่
4. ถ้าไม่ตรงกัน การ์ดจะกลับหงายลง
5. เล่นจนจับคู่ครบทั้ง 6 คู่

## 🔧 การปรับแต่ง

### เพิ่มหรือแก้ไขการ์ด

แก้ไขไฟล์ `cards.json`:

```json
{
  "cards": [
    {
      "id": 1,
      "type": "text",
      "content": "ชื่อพลังงานภาษาไทย",
      "englishName": "English Name",
      "pairId": 1
    },
    {
      "id": 2,
      "type": "image",
      "image": "images/รูปภาพ.png",
      "pairId": 1
    }
  ]
}
```

**หมายเหตุ:** `pairId` ของการ์ดที่จับคู่กันต้องมีค่าเดียวกัน

### เปลี่ยนรูปด้านหลังการ์ด

แก้ไขใน `index.html` ส่วน CSS:

```css
.card-back {
    background-image: url('images/cover.png');
}
```

## 🎨 การปรับแต่งสี

สามารถแก้ไขสีใน `index.html` ส่วน `<style>`:

```css
/* เปลี่ยนสีพื้นหลัง */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* เปลี่ยนสีการ์ด (ถ้าไม่ใช้รูป cover.png) */
.card-back {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

## ⚠️ หมายเหตุ

- ต้องรันผ่าน Web Server หรือ Local Server (ไม่สามารถเปิดไฟล์ HTML โดยตรงได้เนื่องจากใช้ fetch)
- สามารถใช้ Python: `python -m http.server 8000`
- หรือใช้ VS Code Live Server Extension

## 🚀 วิธีเปิดเกม

เนื่องจากใช้ `fetch()` ต้องรันผ่าน Web Server:

```bash
# วิธีที่ 1: Python
python -m http.server 8000

# วิธีที่ 2: PHP  
php -S localhost:8000

# วิธีที่ 3: VS Code Live Server Extension
```

จากนั้นเปิด `http://localhost:8000` ในเบราว์เซอร์
